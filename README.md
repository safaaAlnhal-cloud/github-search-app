# GitHub User Search App

## 📌 Overview
React app to search GitHub users and display profile info.

## 🚀 Features
- Search GitHub users
- Dark / Light mode
- Custom hooks (useTheme, useGitHubUser, useDebounce)
- Debounced search
- Loading & error handling

## 🧠 Tech Used
- React
- JavaScript
- CSS
- Git & GitHub
