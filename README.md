# GitHub User Search App

## 📌 Overview

A React application that allows users to search for GitHub profiles and display their information in a clean UI with dark mode support.

---

## 🚀 Features

- Search GitHub users by username
- Display user profile (name, bio, avatar, repos, followers, following)
- Dark / Light mode toggle
- Custom hooks for better code structure
- Debounced search to reduce API requests
- Loading and error states handling
- Responsive design

---

## 🧠 Custom Hooks

- useTheme → manages light/dark mode
- useGitHubUser → handles API logic and state
- useDebounce → optimizes search input
